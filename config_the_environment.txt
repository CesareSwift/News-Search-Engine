1. 网上搜一下那个flask教程，在里面找一下有一个搭建虚拟python环境的东西，然后照着弄一下，很简单，就是一个pip install
2. 激活虚拟python环境， 我的是运行env/Scripts/activate
3. 配置环境变量，编辑文件.env， 在这个文件里写一句FLASK_APP=hello.py
4. 安装相关的工具包，可以照着requirement.txt一个一个pip install， 也可以直接flask run，提示缺啥就补啥
5. 在工程根目录下，输入命令flask run
6. 然后会提示运行中，有一个网址，叫什么5000.记得在后面加上/search，我得是http://127.0.0.1:5000/search，不知道这个会不会变 用浏览器打开即可
